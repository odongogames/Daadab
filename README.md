# Dadaab Delivery 🚛💧
Dadaab Delivery is a 2.5D arcade-style humanitarian simulation developed for the QS Impact Challenge 2026. Built with Unity 6, the project aims to raise awareness about the critical water logistics challenges in the Dadaab Refugee Complex, Garissa County, Kenya.

## 📖 Project Overview
In March 2026, Northern Kenya is facing a severe climate-induced drought. This project simulates the high-stakes "last-resort" intervention of emergency water trucking. Players navigate a water tanker through a stylized environment to deliver essential supplies to Daadab, bridging the gap between technical humanitarian logistics and public awareness.

## 🛠 Tech Stack
Game Engine: Unity 6000.3.10f1

Language: C#

Platform: WebGL (Mobile-compatible)

Visuals: 2.5D perspective 

## 🚀 Key Features
Arcade Car Controller: Customized physics-based tanker movement optimized for browser interaction.

Narrative Driven: Character-led dialogue that contextualizes the infrastructure failures in North Eastern Kenya.

Typographic Branding: A minimalist UI designed to maintain a professional, yet urgent tone.

Environment Design: Low-poly representation of the Garissa landscape, including indigenous flora and refugee camp layouts.

## 🏗 Setup & Installation
Clone this repository: git clone https://github.com/odongogames/Daadab.git

Open the project in Unity 6000.3.10f1.

To test the WebGL build locally, ensure you are using a local server like XAMPP to avoid CORS and compression errors.

Target Platform: Switch to WebGL in Build Settings.

## 🌍 Humanitarian Context
This project was developed as a direct response to the SDG 6 (Clean Water and Sanitation) crisis in the Daadab Refugee Complex. By gamifying the logistics of water distribution, we highlight the strain on strategic boreholes and the necessity of sustainable climate resilience in East Africa.
